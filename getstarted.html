Get Started

This is a remake of the original service by Gellatio. Thanks to him for creating the version before this. Please read the Terms Of Use before continuing.

1. Apply for a key here. (Please note you MUST be in Aurora Community before applying, join at https://discord.io/aurora)

2. Download the ShareX template here.

3. Open it in a text editor (such as Notepad++) and change INSERT_KEY_HERE to your key.

4. Apply the file to ShareX.
